使用浏览器扩展Print Edit WE优化网页打印效果

Jul 9, 2019

背景

平时阅读到优秀的技术文章，习惯使用Chrome的打印功能将网页归档为PDF文件，有三个目的：

  1. 将好文章归档保存，防止链接失效
  2. 方便在iPad上阅读
  3. 方便做批注

但多数网站对Print Stylesheet的支持都不够好，导致打印效果非常一般。 本文推荐一款好用的Chrome扩展Print Edit WE，可以很方便地优化打印效果，如下图所示：

改进建议

Print Edit WE已经很好用了，但有如下可以改进的地方：

  1. 另存为的PDF文件中，无法像普通PDF文件一样，对文字进行选中、飘黄等操作，这应该是Chrome本身的问题
  2. 调整选中区域样式时，控件操作不太方便，比如不能直接设置margin属性为一个指定值
  3. 是否可以预置一些效果比较好的Print Stylesheet?
